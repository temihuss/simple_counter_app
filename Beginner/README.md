The counter increases each time the **Increment** button is clicked.

This project helped me practice **basic JavaScript concepts such as variables, functions, and DOM manipulation**.

## About the Project

The **People Counter App** is a small interactive webpage that allows a user to click a button to increase a counter.

The number displayed on the page updates every time the button is clicked. This project helped me understand how **JavaScript interacts with HTML elements**.

---

## Features

* Simple and clean user interface
* Increment button to increase the count
* Live update of the counter
* Background styling using CSS

---

## How It Works

1. The webpage displays a counter starting at **0**.
2. When the **Increment** button is clicked, a JavaScript function runs.
3. The function updates the number shown on the page.
4. The new value is displayed instantly.

---

## Technologies Used

* **HTML** – structure of the webpage
* **CSS** – styling and layout
* **JavaScript** – functionality and interactivity

---

## How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in your web browser
4. Click the **Increment** button to increase the counter
